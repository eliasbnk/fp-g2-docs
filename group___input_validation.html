bool GetUserConfirmation ( const tString &  prompt)

Validates and processes yes/no input from the user.

Parameters
prompt- The message displayed to the user as a prompt.

Prompts the user for a yes/no response and validates the input. It accepts variations such as 'Y', 'YES', 'N', and 'NO' (case-insensitive).

Returns
'Y' if the user enters 'Y' or 'YES', 'N' if the user enters 'N' or 'NO'.
